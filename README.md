# Installation App
## Prerequisites

Before running the project, make sure you have:

- Node.js (v18 or higher)
- npm or yarn
- A Supabase account

---

## Setup Instructions

### 1️⃣ Create Supabase Project & Get Keys

1. Go to **https://supabase.com**
2. Create a new project
3. Navigate to **Project Settings → API**
4. Copy the following:
   - Project URL
   - Anon public key

You will need these in the environment variables.
